Stellar Winds

Stellar Winds is a real-time strategy game inspired by Homeworld, Battlestar Galactica and Star Wars: Empire at War.
The game is entirely procedurally generated baring VFX and ship models.
All assets are created by me unless mentioned otherwise.
All code is authored by me unless mentioned otherwise.
